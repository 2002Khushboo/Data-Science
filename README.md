# Data Science Projects

This repository contains a curated collection of data science and machine
learning projects implemented using Python and Jupyter notebooks.

Each project focuses on a real-world dataset and follows a structured
workflow including data preprocessing, exploratory data analysis,
modeling, and evaluation.

## Projects

- **IMDB SQLite Analysis**
  - Data analysis using SQLite and Python to extract insights from movie datasets

- **Load Prediction**
  - Regression-based modeling to predict load using historical data

- **Market Basket Analysis**
  - Association rule mining to identify frequently co-occurring items

- **Uber Data Analysis**
  - Exploratory analysis of ride data to understand demand patterns

- **Walmart Sales Analysis**
  - Sales trend analysis and forecasting using historical retail data

- **Sentiment Analysis**
  - NLP-based sentiment classification on textual review data
